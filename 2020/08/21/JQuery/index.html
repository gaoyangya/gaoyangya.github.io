二.内容

1.JQuery概述

1.1 JavaScript库

JavaScript库,即library,是一个封装好的特定的集合(方法和函数).从封装一大堆函数的角度理解库,就是在这个库中,封装了很多预先定义好的函数在里面,比如动画animate,hide,show,比如获取元素等.

简单理解:就是一个js文件,里面对我们原生js代码进行了封装,存放到里面.这样我们可以快速高效的使用这些封装好的功能.

比如JQuery,就是为了快速方便的操作DOM,里面基本都是函数(方法).为了相区别,jquery称为js库,以前学的js称为原生的js

常见的js库

  • JQuery
  • Prototype
  • YUI
  • Dojo
  • Ext js
  • 移动端的zepto

这些库就是对原生js的封装,内部就是用js实现的

1.2 JQuery概念

JQuery是一个js库,Jquery的作用就是：write less,do more。写的更少，做的更多。

JQuery中j是JavaScript;Query是查询.JQuery意思就是查询js,把js中的Dom操作做了封装,我们可以快速的查询使用里面的功能

2.JQuery基本使用

2.1 JQuery的下载

JQuery官网

  1. 进官网找到要用的JQuery版本(工作中建议用生产版本,学习的时候建议用开发版本)

  1. 将选好的版本中的内容ctrl+c,然后在idea里面ctrl+v

  1. 建一个html文件,引入JQuery文件

2.3 JQuery的入口函数

  1. 等着页面DOM加载完毕再去执行js代码 (传统的写法)

    $(document).ready(function() {
	.....//此处是页面DOM加载完成后要进行的操作
})

  2. 等着页面DOM加载完毕再去执行js代码 (推荐)

    $(function(){
    .....//此处是页面DOM加载完成后要进行的操作
})
  • 等着DOM结构渲染完毕即可执行内部代码,不必等到所有外部资源加载完成.JQuery帮我们完成了封装
  • 相当于原生js中的DOMContentLoaded
  • 不同于原生js中的load事件是等页面文档,外部js文件,css文件,图片加载完毕才执行内部代码

2.4 JQuery的顶级对象$

  1. $是JQuery的别称,在代码中可以使用JQuery代替$,但是为了方便,一般使用$
  2. $是JQuery的顶级对象,相当于原生js中的window.把元素利用$包装成JQuery对象,就可以调用JQuery方法

2.5 JQuery对象和DOM对象及其转换

  1. 用原生js获取的对象就是DOM对象

    var myDiv = document.querySelector("div");

  2. JQuery对象:用JQuery方式获取过来的对象就是JQuery对象.
    本质:通过$把dom元素进行包装

    $("div");  //  $("div")是一个JQuery对象

  3. JQuery对象只能使用JQuery方法,dom对象则只能使用原生的js属性和方法

DOM对象与JQuery对象可以相互转换

因为原生js比JQuery更大,原生的一些属性和方法JQuery没有给我们封装,要想使用这些属性和方法需要把JQuery对象转换为DOM对象才能使用.

  1. DOM对象转换成JQuery对象:
    (1)直接把标签转换为JQuery对象(加引号)$("标签")

    <vedio src="mov.mp4"></vedio>
$("vedio");//加引号

    (2)把dom对象转换为JQuery对象(不能加引号):$(DOM对象)

    <vedio src="mov.mp4"></vedio>
var myVedio = document.querySelector("vedio");
$(myVedio);

  2. JQuery对象转换成DOM对象

    • $("标签")[index]
    • $("标签").get(index)

    index是索引,从零开始.当$(“标签”)里面有多个相同标签的时候来用于区分.不可省略

    <vedio src="mov.mp4"></vedio>
  $("vedio")[0].play();
  $("vedio").get(0).play();

3.JQuery选择器

3.1 JQuery基础选择器

原生js获取元素方式很多,很杂,而且兼容情况不一致,因此JQuery给我们做了封装,使获取元素统一标准

$("选择器")   //里面选择器直接写css选择器即可,但是要加引号
名称 用法 描述
ID选择器 $(“#id”) 获取指定id的元素
全选选择器 $(“*”) 匹配所有元素
类选择器 $(“.class”) 获取同一类class的元素
标签选择器 $(“div”) 获取同一类标签的所有元素
并集选择器 $(“div,p,li”) 获取多个元素
交集选择器 $(“li.current”) 交集元素

3.2 JQuery层次选择器

名称 用法 描述
子代选择器 $(“ul>li”) 使用>号,只仅仅获取亲儿子层次的元素
后代选择器 $(“ul li”) 使用空格,代表后代选择器,获取ul下的所有li元素,包括孙子,,,,

JQuery设置样式

$("div").css("属性","值")

3.3. 隐式迭代:star::star:

遍历内部DOM元素(以伪数组的形式存储)的过程就叫做隐式迭代

简单理解:隐式迭代就是把匹配的所有元素内容进行遍历循环,而不用我们再去循环

$("div").css("属性","值")
//隐式迭代就是把匹配的所有元素内容进行遍历循环,给每一个元素添加css这个方法

3.4 JQuery筛选选择器

语法 用法 描述
:first $(“li:first”) 获取第一个li元素
:last $(“li:last”) 获取最后一个li元素
:eq(index) $(“li:eq(2)”) 获取到的li元素中,选择索引号为2的元素,索引号index从0开始
:odd $(“li:odd”) 获取到的li元素中,选择索引号为奇数的元素
:even $(“li:even”) 获取到的li元素中,选择索引号为偶数的元素

4 JQuery样式操作

4.1 操作css方法

JQuery可以使用css;也可以操作类,修改多个样式方法来修改简单元素样式.

  1. 查看样式:参数只写属性名,则返回属性值

    $(this).css("color");

  2. 修改样式:参数是属性名,属性值,逗号分隔,是设置一组样式,属性必须加引号,值如果是数字可以不用加单位跟引号

    $(this).css("color","red");

  3. 修改多组样式:参数是对象形式**(所以要有一个大括号{})**,方便设置多组样式.属性名和属性值用冒号隔开,属性可以不用加引号,如果值不是数字,则需要加引号

    $(this).css({"color":"red","font-size":"20px"});

         });
     //    鼠标离开
     //    hide() 隐藏元素
         $(".nav>li").mouseout(function (){
         	$(this).children("ul").slideUp(200);
         })
     })

<script> 
//1.事件切换 hover就是鼠标经过和离开的复合写法

            $(this).children("ul").slideDown(200);

            $(this).children("ul").slideUp(200);        })
</script>
  1. 进一步化简
<script>
// 2.事件切换 hover 如果只写一个函数,那么鼠标经过和离开都会触发这个函数

        $(this).children("ul").slideToggle();

5.2.2动画队列及其停止排队的方法

  1. 动画或效果队列

动画或效果一旦触发就会执行,如果多次触发,就造成多个动画或者效果排队执行.
譬如:在下拉菜单中,不论你用显示隐藏(show(),hide())还是滑动,,,当你快速在多个菜单上面来回切换点击的时候,二级菜单就会一段时间内不停的显示

  1. 停止排队
stop()
  1. stop()方法用于停止动画或效果
  2. 注意:stop()写到动画或效果前面,相当于停止结束上一次的动画(不管你触发多少次,我只执行最新的一次)

必须要写在动画或者效果前面!!!!
